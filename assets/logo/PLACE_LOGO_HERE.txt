Letakkan file logo resmi TNC di sini dengan nama persis: tnc-logo.png

Seluruh halaman (sidebar, header, halaman login/register, favicon) sudah
mereferensikan path ini secara otomatis:

  assets/logo/tnc-logo.png

Jika file belum ada, elemen <img> akan otomatis disembunyikan (melalui
atribut onerror di setiap halaman) sehingga layout tidak rusak — namun
untuk tampilan production yang lengkap, pastikan file ini diunggah.

Format yang disarankan: PNG dengan latar transparan, minimal 256x256px,
agar tetap tajam di seluruh ukuran (sidebar 34px, halaman login 40px,
favicon browser).
